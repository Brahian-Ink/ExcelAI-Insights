# Excel AI Insights

Upload an Excel file (.xlsx), preview normalized data, and generate AI-powered insights (data quality issues, trends, and recommendations).

## Tech Stack
- Backend: ASP.NET Core Web API (C#)
- Frontend: JavaScript (React planned)
- AI: LLM API (via backend)

## MVP Features
- Upload .xlsx
- Preview parsed rows + detected columns
- AI insights based on a sampled subset of rows (no database)

## Roadmap
- Add exportable insight reports (Markdown/PDF)
